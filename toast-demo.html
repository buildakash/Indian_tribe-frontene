Basic Toast Types

Click the buttons below to see different types of toast notifications:

// Basic usage window.toast.success('Operation completed successfully!', 'Success'); window.toast.error('Something went wrong', 'Error'); window.toast.warning('Please check your input', 'Warning'); window.toast.info('Processing your request...', 'Info');

Advanced Toast Options

Customize toast behavior with advanced options:

// Advanced options window.toast.show({ type: 'success', title: 'Custom Title', message: 'Custom message with advanced options', duration: 10000, closable: false, onClose: () => console.log('Toast closed!') });

Toast Management

Control multiple toasts and clear them as needed:

// Toast management window.toast.clear(); // Clear all toasts window.toast.clearType('error'); // Clear only error toasts window.toast.clearType('success'); // Clear only success toasts

Features & Benefits

  • Beautiful Design: Modern, gradient-based design with smooth animations
  • Multiple Types: Success, Error, Warning, and Info with distinct colors
  • Auto-dismiss: Configurable duration with visual progress bar
  • Responsive: Adapts to all screen sizes automatically
  • Accessible: ARIA labels and keyboard navigation support
  • Customizable: Easy to configure duration, titles, and callbacks
  • Lightweight: Minimal impact on page performance
  • Cross-browser: Works on all modern browsers

Integration with Error Handler

See how toasts work seamlessly with the error handling system:

// Error handling integration try { const result = await apiCall(); window.toast.success('Success!', 'Operation Complete'); } catch (error) { window.errorHandler.showErrorToast(error, 'Context'); }
